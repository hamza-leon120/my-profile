## ✨ Features

- **Fully Responsive** — Optimized for mobile, tablet, and desktop
- **Smooth Scroll Animations** — Sections fade in on scroll using the `IntersectionObserver` API
- **Sticky Header** — Hides on scroll and reappears with a highlighted style at key scroll positions
- **Mobile Navigation Menu** — Animated hamburger menu with smooth height transition
- **Auto-Sliding Project Carousel** — Projects slide automatically every 4 seconds, resets timer on manual interaction
- **Image Fullscreen Modal** — Click any project image to view it fullscreen; close by clicking the X button or anywhere outside the image
- **Scroll Progress Indicator** — A circular `conic-gradient` widget shows the current scroll percentage through the page, fades in after 300px
- **Scroll-to-Top Button** — Appears after 300px of scrolling with a smooth fade transition
- **Scroll Indicator** — Animated mouse + wheel + diamond visual between sections
- **Brand-Accurate Skill Icons** — Each technology displayed with its official brand color
- **Font Awesome v7** — Latest icon library including the Tailwind CSS brand icon

## 📂 Project Structure

```
my-profile/
├── .github/
│   └── workflows/        # GitHub Actions CI/CD pipeline
├── Compiled-HTML/        # Pre-compiled HTML output
├── img/                  # Images and assets
├── include/              # Pug includes/partials
├── scss/
│   ├── layout/           # Layout styles
│   ├── mixin/            # SCSS mixins (responsive breakpoints)
│   ├── var/              # SCSS variables (color palette)
│   └── style.scss        # Main SCSS entry point
├── index.html            # Compiled HTML file
├── index.pug             # Pug template source
├── main.js               # JavaScript logic
├── .gitignore
├── LICENSE
└── README.md
```

## ⚙️ Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/hamza-leon120/my-profile.git

# 2. Navigate into the project directory
cd my-profile

# 3. Open index.html in your browser
open index.html
```

> **Note:** To edit the source files, you'll need a Pug/SCSS compiler like [Prepros](https://prepros.io/) or a build tool like Gulp to compile `.pug` → `.html` and `.scss` → `.css`.
